# proyecto-mineria-datos
Proyecto Final de Minería de Datos - Predicción de Tiros de la NBA (Maestría en Ciencias de la Computación)

# Predicción de Efectividad de Tiros en el Básquetbol (NBA)

Este repositorio contiene el proyecto final para el curso de Minería de Datos, desarrollado bajo los principios de **Ciencia 2.0** (código abierto, datos públicos y experimentos reproducibles).

## Estructura del Repositorio
- `data/`: Contiene el set de datos histórico de tiros.
- `notebook.ipynb`: Notebook principal con el flujo de minería de datos (EDA, Preprocesamiento, Modelado y Evaluación).
- `requirements.txt`: Librerías necesarias para ejecutar el proyecto.

## Ejecución
El análisis fue diseñado para ser ejecutado directamente en **Google Colab**.
